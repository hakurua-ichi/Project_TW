# 게임 소개
유니티 짱이 2D의 세계로 떨어지면서 자아를 탐색하기 위한 여정을 그려냈습니다.
3D로 제작된 맵을 2D의 시점에서 진행하게 되며 **카메라 회전**과 **맵 회전**의 기믹을 활용하여 세상을 탐험해보세요.
**세상의 축**과 **그 축에 남겨진 꿈과 자아**를 느껴보세요.
